# MicroNutri — Precision Nutrition AI
> Scan. Analyze. Nourish. Instant micronutrient intelligence in your pocket.

## Overview
MicroNutri is a camera‑first, AI‑powered nutrition companion that delivers instant micronutrient and macronutrient insights through a sleek, futuristic mobile interface.

It combines:
- Camera-based food scanning
- AI nutrient analysis
- Daily nutrition tracking
- Personalized coaching
- A dark, glassy, premium UI

## Core Features (Planned)
- Camera-first food scanning
- AI-powered nutrient breakdown (micro + macro)
- Daily dashboard with streaks, calories, macros
- AI-verified meals and match confidence
- Personalized AI coach and deficiency insights
- Goal-based recommendations (fat loss, muscle gain, longevity)

## Tech Stack
- Vite (frontend tooling)
- Capacitor (Android/iOS builds)
